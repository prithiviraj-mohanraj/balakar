# BALAKAR SPARKLERS WEBSITE MASTER BRIEF

Build a world-class premium manufacturer website for Balakar Sparklers.

This website should look like a modern 2026 premium product brand website rather than a traditional fireworks website.

## Design Direction

Use:

* White background
* Light premium theme
* Luxury visual design
* Modern spacing
* Professional typography
* Mobile first design
* Responsive design
* Elegant micro animations
* Smooth scrolling
* Glassmorphism where appropriate
* Framer Motion animations

Visual Inspiration:

* Apple
* Stripe
* Framer
* Linear
* Vercel

Avoid:

* Red background websites
* Old Sivakasi style layouts
* Flashing animations
* Cluttered product tables
* Outdated gradients

## Branding Rules

Keep original logo exactly as provided.

Keep Phoenix trademark exactly as provided.

Keep Green Fireworks certification exactly as provided.

Do not redesign or alter branding assets.

## Image Processing Rules

Use uploaded catalog images as references.

Create premium website imagery from the catalog assets.

If any certification image contains background text, watermark text, company text, or unwanted labels, clean the image while preserving:

* Certification logo
* Certification number
* Certification artwork
* Original colors

Do not modify the official logo designs.
